# LSS Clinic — Лаборатория современной стоматологии

Современная информационная система для стоматологической клиники, состоящая из FastAPI бэкенда и Next.js фронтенда.

## Структура проекта

- `/backend` — API на Python (FastAPI), обработка заявок.
- `/frontend` — Веб-интерфейс на Next.js (App Router).

## Как запустить проект

Для полноценной работы проекта необходимо запустить оба компонента.

### 1. Запуск Бэкенда

Убедитесь, что у вас установлен Python 3.9+.

```bash
cd backend
# Создайте виртуальное окружение (рекомендуется)
python3 -m venv venv
source venv/bin/activate  # Для macOS/Linux
# venv\Scripts\activate  # Для Windows

# Установите зависимости
pip3 install -r requirements.txt

# Запустите сервер
python3 -m uvicorn main:app --reload
```
Бэкенд будет доступен по адресу: `http://localhost:8001`

### 2. Запуск Фронтенда

Убедитесь, что у вас установлена Node.js (версия 18+).

```bash
cd frontend

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
```
Фронтенд будет доступен по адресу: `http://localhost:3000`

## Основные возможности

- **Адаптивный дизайн**: Сайт корректно отображается на всех устройствах.
- **Онлайн-запись**: Форма на главной странице отправляет данные на бэкенд.
- **SEO**: Настроены метатеги и заголовки для всех страниц.
- **SPA навигация**: Быстрые переходы между страницами без перезагрузки.
